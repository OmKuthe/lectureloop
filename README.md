# LectureLoop

LectureLoop is a MERN-based web application that converts lecture notes into interactive quizzes for immediate classroom revision and engagement tracking.

## Features
- Upload lecture notes (PDF/DOCX/TXT)
- Auto-generate 10 MCQs
- 15-second timer per question
- One attempt per student
- Teacher performance dashboard

## Tech Stack
- MongoDB
- Express
- React
- Node.js
